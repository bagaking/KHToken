# KHToken

## Prepare

#### enviroment:
1. install node
2. update npm to latest
3. install yarn

#### repo:
1. clone the repo
2. ```cd ./KHToken``` 

## Test
1. ```yarn install```  
2. ```npm run test```  

## Deployment
